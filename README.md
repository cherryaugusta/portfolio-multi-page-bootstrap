# Cherry Augusta — Developer Portfolio

**Professional multi-page portfolio website** of Cherry Augusta, a London-based Software Engineer specializing in **Python, Django, TypeScript, React, Angular, PostgreSQL, Redis, Celery, Docker**, backend APIs, full-stack applications, and reviewable AI-assisted workflow systems.

## Features

- **Modern multi-page design** built with Bootstrap 5 + custom CSS
- Fully responsive with excellent mobile experience
- Strong accessibility (skip link, semantic HTML, ARIA labels)
- SEO optimized (meta tags, Open Graph, Twitter cards, canonical URLs)
- Fast loading with lazy loading and optimized assets
- Professional dark/light aesthetic with premium UI components
- GitHub Pages deployment via GitHub Actions
- Current year auto-updated in footer

## Pages

- **Home** (`index.html`) — Hero, strengths, flagship projects, core stack
- **Projects** (`projects.html`) — Detailed flagship, featured, Angular, and supporting projects
- **CV** (`cv.html`) — Professional summary, skills, experience, education, certificates
- **About** (`about.html`) — Engineering approach and background
- **Contact** (`contact.html`) — Contact details and preferences

## Tech Stack

- **HTML5**, **CSS3** (custom design system)
- **Bootstrap 5.3** (responsive grid, navbar, components)
- **Vanilla JavaScript** + jQuery-ready structure
- **Google Fonts** (Inter)
- **GitHub Pages** + GitHub Actions CI/CD
